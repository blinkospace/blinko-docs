# Hotkey Settings

Quickly open Blinko notes and AI assistant from any application.

## Quick Setup

1. Open Settings â†’ Desktop & Hotkeys
2. Enable hotkey functions
3. Click "Record" button, press your desired key combination

## Default Hotkeys

- **Quick Note**: `Shift + Space`
- **Quick AI**: `Alt + Space`

## Supported Keys

- **Modifiers**: Ctrl/Cmd, Alt, Shift
- **Letters/Numbers**: A-Z, 0-9
- **Function Keys**: F1-F12
- **Special Keys**: Space, Tab, Esc

## FAQ

**Hotkeys not working?**
1. Check if enabled
2. Restart Blinko
3. Check for software conflicts

**Reset hotkeys**
Click the refresh button next to settings
