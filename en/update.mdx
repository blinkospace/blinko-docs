## Bash Script Update


## Watchtower
<Warning>Do not add **postgres** to watchtower or your data will be lost</Warning>
<AccordionGroup>
  <Accordion icon="square-terminal" title="Watchtower" defaultOpen>
```bash
docker run -d \
 --name watchtower \
 --network=bridge \
 --restart=always \
 -v /var/run/docker.sock:/var/run/docker.sock \
 -w / \
 containrrr/watchtower blinko-website --cleanup -i 3600 --run-once
```
  </Accordion>
</AccordionGroup>
