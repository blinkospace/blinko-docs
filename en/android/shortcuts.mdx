# Quick Launch

Use Android app shortcuts to quickly launch specific Blinko features.

## Available Shortcuts

### Quick Note
- **Function**: Directly open new note interface
- **Icon**: Note icon
- **Use Case**: Quickly capture ideas

### Voice Recording
- **Function**: Directly open voice recording interface
- **Icon**: Microphone icon
- **Use Case**: Voice note recording

## How to Access

### Method 1: Long Press App Icon
1. Find Blinko on home screen or app drawer
2. Long press the app icon
3. Select the shortcut you need

### Method 2: Add to Home Screen
1. Long press Blinko app icon
2. Drag shortcut to home screen
3. Tap desktop shortcut directly to use

## Shortcut Features

### Quick Note
- Skip main interface, go directly to editor
- Auto-focus input field
- Support various content types

### Voice Recording
- Directly start recording function
- Auto-convert to text (requires AI config)
- Support long-duration recording

## Usage Tips

- **Desktop Shortcuts**: Add frequently used features to home screen
- **Quick Switch**: Switch between different features via shortcuts
- **Efficiency Boost**: Avoid time spent navigating within the app

## System Requirements

- Android 7.0 (API 25) and above
- Launcher that supports app shortcuts

## FAQ

**Don't see shortcuts?**
1. Ensure Android version is supported
2. Check if launcher supports app shortcuts
3. Reinstall Blinko app

**Shortcuts not working?**
1. Restart the app
2. Clear app cache
3. Update to latest version
