## Watchtower
<Warning>不要将**postgres**添加到watchtower，否则您的数据将丢失</Warning>
<AccordionGroup>
  <Accordion icon="square-terminal" title="Watchtower" defaultOpen>
```bash
docker run -d \
 --name watchtower \
 --network=bridge \
 --restart=always \
 -v /var/run/docker.sock:/var/run/docker.sock \
 -w / \
 containrrr/watchtower blinko-website --cleanup -i 3600 --run-once
```
  </Accordion>
</AccordionGroup>
