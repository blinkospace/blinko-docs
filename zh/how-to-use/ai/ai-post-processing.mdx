# AI后处理

AI后处理功能可以在创建或更新笔记后自动使用AI分析内容，生成摘要、标签或执行其他分析操作。

<img className="rounded-2xl" src="/images/2025-09-19-16-23-48.png" />

## 启用方法

1. 打开设置 → AI Settings
2. 找到"AI后处理"部分
3. 开启"启用AI后处理"开关

## 处理模式

### 添加为评论
- AI分析结果以评论形式添加到笔记下方
- 不会修改原始笔记内容
- 可以查看AI的分析过程和结果

### 自动添加标签
- AI根据内容自动推荐和添加相关标签
- 帮助更好地组织和分类笔记
- 提高搜索和查找效率

### 两者都执行
- 同时生成评论和添加标签
- 最全面的AI处理方式

## 自定义提示词

### 后处理提示词
设置AI分析笔记时使用的提示词：
```
分析以下笔记内容，推荐合适的标签并提供简短摘要
```

### 标签提示词
专门用于生成标签的提示词：
```
为以下内容推荐3-5个相关标签
```

## 工作流程

1. **创建笔记** - 用户创建新笔记
2. **触发处理** - 系统自动检测到新笔记
3. **AI分析** - 使用配置的提示词分析内容
4. **生成结果** - 根据处理模式添加评论或标签
5. **通知用户** - 发送AI处理完成的通知

## 使用场景

### 内容分析
- 自动总结长文本内容
- 提取关键信息和要点
- 分析情感和主题

### 智能标签
- 根据内容自动分类
- 识别专业领域和话题
- 建立一致的标签体系

### 工作流集成
- 会议记录自动处理
- 学习笔记智能整理
- 项目文档分类管理

## 注意事项

- 处理过程在后台异步执行，不会影响笔记创建速度
- AI分析质量取决于所选模型和提示词设置
- 可以随时关闭功能，已处理的内容不会受影响
- 支持自定义提示词以适应不同使用场景

## 配置建议

### 提示词优化
- 根据个人需求调整提示词
- 明确指定期望的输出格式
- 包含相关的上下文信息

### 模式选择
- 新用户推荐先使用"添加为评论"模式
- 熟悉后可切换到"两者都执行"获得最佳效果
- 标签管理规范的可只使用"自动添加标签"

<Warning>
**重要提醒**：
- AI后处理功能需要先启用AI并配置相关模型
- 处理结果可能需要人工审核和调整
- 建议定期检查AI生成的标签和评论质量
</Warning>
